# PWAアイコンの作成方法

以下のサイトでアイコンを生成してください：
https://www.pwabuilder.com/imageGenerator

または、以下の手順で手動作成：

1. 512x512のPNG画像を作成
2. 以下のファイル名で保存：
   - icon-192.png (192x192にリサイズ)
   - icon-512.png (512x512)

アイコンのデザイン案：
- 背景: 青色 (#3b82f6)
- アイコン: 📚 または 📝 の絵文字
- テキスト: なし（シンプルに）

簡易版として、現在はテキストファイルを配置しています。
実際のアイコンファイルを後で追加してください。
